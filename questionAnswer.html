Question Number 1 : Difference between map,filter,for each,find

map is used to transform each element of an array, filter is used to create a new array with only the elements that pass a certain condition, forEach is used to execute a function for each element in an array, and find is used to find the first element in an array that satisfies a certain condition.

Question Number 2 : Difference between local,section,cookies storage e

local storage is used to store larger amounts of data that needs to persist across multiple browser sessions, while session storage is used to store data that needs to be available only during the current browser session. Cookies are used to store small amounts of data that can persist beyond the current browser session and can be accessed by both the client-side and server-side code.

Question Number 3 : what is event loop

The event loop works by constantly checking a queue of pending events. When an event occurs, it is added to the queue, and the event loop begins processing the events in the queue. Each event is processed in turn, and the appropriate code is executed based on the type of event.
